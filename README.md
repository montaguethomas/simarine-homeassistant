# Simarine Home Assistant

Custom Home Assistant integration for Simarine and compatible devices.


## Features

- Automatic sensor discovery
- Native Home Assistant entities
- UI configuration (no YAML required)
- Uses simarine-python library

## Installation (HACS)

1. Open HACS → Integrations
2. Add custom repo:
   https://github.com/montaguethomas/simarine-homeassistant
3. Install
4. Restart Home Assistant
5. Add Integration → "Simarine"

## Installation (Manual)

Copy `custom_components/simarine` into your HA config directory.

## Requirements

- [simarine-python](https://github.com/montaguethomas/simarine-python)
